## Getting Started

### Prerequisites

- Python 3.8+
- PostgreSQL
- Docker

### Installation

1. **Clone the repository:**

    ```sh
    git clone https://github.com/10-academy-w5-group-2/llm-amharic.git
    cd llm-amharic
    ```

2. **Run the project with Docker:**

    ```sh
    docker-compose up
    ```
